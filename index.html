Rummy Assistant (Two-Decks + Missing Cards, Fixed)

Enter exactly 13 cards total. For each suit (♣, ♦, ♥, ♠), type your ranks as a continuous string (no commas/spaces), using:
`A 2 3 4 5 6 7 8 9 T J Q K` (where **T** = “10”). Then choose your Wild-card rank (or leave blank), and specify how many pure Jokers (JOK) you have (0–2).

Note on Two Decks: You may see the same rank-suit twice (e.g. two “8♣”). For any set, duplicates of the same suit do not count twice in that set or run. All grouping logic (pure/impure sets, pure/impure sequences) and leftover-point logic remain as before.

New Feature—Missing Cards for Pure Sequences:
For each suit, this lists exactly which ranks you would need in order to complete any 2-card (or larger-gap) into a pure sequence of length ≥ 3. - Example: If you have 6♥ and 7♥, you need 5♥ and 8♥. - If you have 4♣ and 6♣, you need 5♣. - If you have 4♣ and 7♣, you need 5♣ and 6♣.
