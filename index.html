Rummy Assistant (Pure Sequences Excluding Wild When Unneeded)

Enter your cards by suit, choose the wild card rank, and specify how many pure Jokers (JOK) you have (0–2). Now, when forming pure sequences in each suit, the assistant will:

  • First attempt to build runs (length ≥ 3) using only non‐wild cards.
  • Only if no run ≥ 3 exists without the wild, will it include a wild‐rank card as part of a run.
  • This avoids “wasting” the wild in a suit when you can already form a pure run of length ≥ 3 without it.
The assistant then:
  1. Lists all pure sequences found under these rules (length ≥ 3). Removes them.
  2. Lists all pure sets (three or more of same rank). Removes them.
  3. Repeatedly finds the optimal impure sequence (length ≥ 3) minimizing leftover points—using Jokers and any remaining wilds—removing each.
  4. Any cards left whose rank ≠ wild‐rank are “leftover” and shown; wild‐rank and Joker leftovers are excluded (0 points).'
  5. Points for shown leftover: A/10/J/Q/K=10; 2–9=face value; capped at 80.
